# Unit testing
unit testing is a testing technique using which individual modules are tested if there are any issue by developer himself
the main aim is to isolate each unit of system to identify, analyse and fix the defects

## Advantages of unit testing
1. Reduces defects in the newly developed features and reduces bug when changing the existing functionality
2. Reduces the cost of testing as defects are captured in the very early phase
3. Improve design and batter refactoring of code
